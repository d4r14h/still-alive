# Terminal version of the 'Still Alive' song from the ending of the Portal game

Dependencies:
-----------------

* ncurses shared library
* aplay utility from the ALSA tools

How to compile:
-----------------

```
$ cd still-alive
$ make
```

How to use:
-----------------

```
$ ./still-alive
```
